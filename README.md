# laravel_mini01

## ファイル構成の学習メモ
├── app/               # アプリケーションロジック（コントローラ、モデルなど）<br>
├── bootstrap/         # フレームワークのブートストラップファイル<br>
├── config/            # 設定ファイル<br>
├── database/          # データベース関連ファイル<br>
├── public/            # Webサーバーのルートディレクトリ<br>
├── resources/         # ビューや言語ファイル<br>
├── routes/            # ルーティングファイル<br>
├── storage/           # キャッシュやログ<br>
├── tests/             # テストファイル<br>
└── .env               # 環境変数ファイル<br>
